# Monetary Policy Challenge

An interactive HTML/CSS/JavaScript classroom game where students act as a country's central banker.

## Learning goal

Students make monetary policy decisions round-by-round and learn how each choice affects:
- Inflation
- Unemployment
- GDP growth
- Money supply
- Economic confidence

The objective is to keep the economy stable: avoid recession while preventing inflation from getting out of control.

## Live site

`https://atomlinson.github.io/Monetary-Policy/`

## Run locally

Open `index.html` directly in your browser, or run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
